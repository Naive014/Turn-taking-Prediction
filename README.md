# Turn-taking Prediction
# Классификация завершенности мысли пользователя на уровне последовательности сообщений

## 📌 Описание проекта

Этот проект посвящён задаче бинарной классификации текстовых сообщений с использованием моделей из библиотеки `transformers`. Основная цель — определить, является ли последовательность сообщений сигналом к остановке (`STOP`) или нет (`NOTSTOP`).

Особенность проекта — преобразование числовых признаков (time_diff, is_reply) в токены и работа с ними как с текстом

---

## Основная информация

- Загрузка и обработка данных в формате `.jsonl`
- Инженерия признаков на основе временных интервалов
- Генерация специальных токенов:
  - Временные интервалы (`[TINY]`, `[SMALL]`, `[MEDIUM]`, `[LARGE]`, `[ELARGE]`)
  - Начальное состояние (`[START_HOT]`, `[START_WARM]`, `[START_COLD]`)
  - Скорость общения (`[VEL_*]`)
- Дообучение трансформерной модели для классификации
- Воспроизводимость результатов за счёт фиксированного seed
